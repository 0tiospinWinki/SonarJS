Because semi-colons at the ends of statements are optional, splitting a chain of function calls over multiple lines makes the code confusing and hard to maintain.

Noncompliant Code Example

doSomething(foo)
(x++)
doSomethingElse(bar)

Compliant Solution

What was the initial purpose of the developer ? 

doSomething(foo)(x++);
doSomethingElse(bar);

or 

doSomething(foo);
x++;
doSomethingElse(bar);
