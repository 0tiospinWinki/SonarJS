By convention, a file that exports only one class should be named for that class. Anything else may confuse maintainers.

Noncompliant Code Example

// file path: myclass.js  -- Noncompliant
class MyClass {
  // ...
}
export default MyClass;

Compliant Solution

// file path: MyClass.js
class MyClass {
  // ...
}
export default MyClass;
